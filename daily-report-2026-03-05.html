主要遇到的问题

响应速度延迟（已解决）

问题：简单天气查询耗时约 10 分钟，用户体验差

根因：每次操作都保存记忆、默认模型未优化、工具调用未批处理

解决：减少记忆保存频率、改用 MiniMax-M2.5 处理简单任务、合并工具调用

跨平台会话隔离（已解决）

问题：Telegram 和 Dashboard 会话/记忆不共享，体验不一致

根因：session.dmScope 配置为"per-channel-peer"

解决：改为"global"配置，重启 Gateway 应用

Dashboard 显示系统消息

问题：Dashboard 显示"Pre-compaction memory flush"等内部系统提示

影响：仅显示问题，不影响功能；Telegram 正常过滤

状态：低优先级，待后续修复 Dashboard 消息过滤
